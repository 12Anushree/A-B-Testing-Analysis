## 📚 A/B Testing Rules
✅ Calculate sample size BEFORE running
✅ Run sanity checks first
✅ Use correct statistical test
✅ Check for novelty effects
✅ Segment your results
✅ "No effect" IS a valid result
✅ Don't p-hack!
